🚀 Novo Projeto: SignalHub APIs

Acabei de lançar o SignalHub APIs, uma plataforma de observabilidade de dados que demonstra como construir sistemas backend confiáveis e transparentes.

🔧 O que faz:
• Integra 3 APIs públicas (clima, moeda, cripto)
• Normaliza dados heterogêneos em sinais unificados
• Monitora freshness e executa quality checks
• Expõe tudo através de um dashboard premium

💡 Por que é diferente:
A maior parte do trabalho de backend é invisível. Pipelines rodam silenciosamente, integrações quebram sem aviso, qualidade erode sem verificações. SignalHub torna cada aspecto do pipeline de dados observável, rastreável e explicável.

🛠️ Stack:
• Backend: FastAPI + SQLAlchemy + APScheduler
• Frontend: Next.js 16 + TypeScript + shadcn/ui
• Database: SQLite (dev) / PostgreSQL (prod)
• Validação: Pydantic + Quality Checks

✅ Status: Totalmente funcional e validado end-to-end

================================================================================
TROUBLESHOOTING
================================================================================

Se der erro de autenticação:
1. Vá em: https://github.com/settings/tokens
2. Clique em "Generate new token (classic)"
3. Marque: repo, workflow, write:packages
4. Copie o token
5. Use o token como senha quando o git pedir

Se der erro "remote origin already exists":
git remote remove origin
git remote add origin https://github.com/BarujaFe1/signalhub-apis.git
